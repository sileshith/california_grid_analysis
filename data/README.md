# Data Access

The raw and processed data files are not included in this repository because of file size and source distribution limits. Large data files are excluded from version control by `.gitignore`.

---

## Raw Data — EIA-930 Balancing Authority Hourly Operations

**Source:** U.S. Energy Information Administration (EIA) Form EIA-930  
**Publisher:** U.S. Energy Information Administration (public domain)  
**Download page:** https://www.eia.gov/electricity/gridmonitor/dashboard/electric_overview/US48/US48

To download the file used in this project:
1. Visit the EIA Grid Monitor download page above
2. Select **Balancing Authority** data
3. Select the time period: **2026 Jan–Jun**
4. Download the file: `EIA930_BALANCE_2026_Jan_Jun.csv`

Place the downloaded file locally at:

```
data/raw/EIA930_BALANCE_2026_Jan_Jun.csv
```

The file is approximately 27 MB.

---

## Processed Data — cleaned_california_grid_data.csv

The cleaned, California-filtered dataset is generated by running the analysis notebook:

```
notebooks/california_grid_analysis.ipynb
```

The notebook performs the following preparation steps:
1. Loads `data/raw/EIA930_BALANCE_2026_Jan_Jun.csv`
2. Renames and standardizes column names
3. Removes rows with missing values in key fields
4. Removes duplicate records
5. Converts UTC timestamps to Pacific Time
6. Filters to California balancing authorities only (BANC, CISO, IID, LDWP, TIDC)
7. Engineers a Stress Index (demand as a percentage of observed peak demand)
8. Classifies hours into review-priority categories (High / Medium / Low)
9. Saves the cleaned output to `data/processed/cleaned_california_grid_data.csv`
10. Saves the top review hours summary to `outputs/top_review_hours_california.csv`

## Why Data Files Are Excluded from Git

- The raw EIA file is 27 MB — well above typical Git repository limits and slow to clone
- The processed file is 3.6 MB — also excluded to keep the repository lightweight
- The EIA data is publicly available and freely reproducible from the source above
- Storing large data files in Git can corrupt repository history and cause push/pull failures on GitHub
